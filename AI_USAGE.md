# AI Usage

## AI Tool Used
- ChatGPT (OpenAI)
- Claudeai (Anthropic)

## Most Useful Prompt 1
"Help me design a clean FastAPI project structure for a leave management system."

How it helped:
It provided a maintainable folder structure and explained the responsibilities of each module.


## Most Useful Prompt 2
"Review my implementation of the 30% leave rule and identify possible edge cases."

How it helped:
It highlighted cases involving weekends, public holidays, and overlapping leave requests that I tested.



## One Case Where AI Got It Wrong

During development, AI suggested an approach for validating overlapping leave requests that did not fully align with the assessment's business rules. After comparing the suggestion with the project brief, I corrected the implementation and added tests to ensure only overlapping approved leave requests were rejected according to my documented design decisions.



## Final Note

AI was used to assist with brainstorming, code review, and documentation. I independently implemented the solution, made the final design decisions, verified the business rules against the project brief, and tested the application before submission.
